# app-courses

### Requisitos

- App de cursos gratuitos
- Aluno pode acessar todos os cursos
- Api em Wordpress
- Um curso pode ter vários módulos
- Um módulo pode ter várias aulas

### Páginas:
- Home
- Cursos 
- Single Curso
- Single Aula
- Sobre o Professos
- Contato
